# FUTURE_ML_01
# 📈 Sales & Demand Forecasting using Linear Regression

## 🚀 Project Overview

This project focuses on forecasting future sales using historical business data with the help of **Machine Learning** techniques.
A **Linear Regression** model is used to analyze sales trends and predict future demand.

The project demonstrates:

* 🧹 Data Cleaning
* 🕒 Time-based Feature Engineering
* 📊 Sales Forecasting
* 📉 Model Evaluation
* 📈 Business-friendly Visualizations

---

# 🎯 Objective

The main objective of this project is to:

✅ Predict future sales using historical data
✅ Analyze business sales trends over time
✅ Support better inventory and demand planning
✅ Build a beginner-friendly forecasting model

# ⚙️ Project Workflow

## 📥 1. Data Collection

* Imported historical sales dataset from Kaggle.

## 🧹 2. Data Cleaning

* Removed missing values
* Converted date columns into datetime format

## 🕒 3. Feature Engineering

Extracted time-based features such as:

* Day
* Month
* Year
* Day of Week

## 🤖 4. Model Building

Used **Linear Regression** algorithm to train the forecasting model.

## 📏 5. Model Evaluation

Evaluated model performance using:

* MAE (Mean Absolute Error)
* RMSE (Root Mean Squared Error)

## 📊 6. Visualization

Generated graphs for:

* Actual Sales
* Predicted Sales
